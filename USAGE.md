# 孩子成长记录表使用说明

## 概述
这是一个用于记录孩子各个阶段生长发育情况的移动端兼容网页应用，参考了7岁以下儿童生长标准。

## 功能特性
- 记录孩子的身高、体重等生长发育数据
- 可视化展示孩子的生长曲线
- 与国家标准进行对比分析
- 响应式设计，兼容移动端和桌面端
- 简洁易用的用户界面
- 数据本地存储，无需注册登录

## 使用方法

### 1. 记录孩子信息
- 在"孩子信息"部分填写孩子的姓名、性别和出生日期
- 这些信息将用于计算孩子的年龄和生长发育对比

### 2. 添加生长记录
- 在"生长记录"部分填写记录日期、身高、体重和头围
- 点击"添加记录"按钮保存数据
- 系统会自动计算孩子的年龄（以月为单位）

### 3. 查看生长曲线
- 在"生长曲线图"部分可以查看孩子的身高、体重和头围变化趋势
- 图表会自动更新以显示最新的数据

### 4. 管理记录
- 在"记录列表"部分可以查看所有已保存的记录
- 可以删除不需要的记录

## 生长标准对比
系统会根据孩子的性别和年龄，自动将身高和体重数据与国家标准进行对比，并在记录列表中显示百分位信息。

## 技术实现
- HTML5
- CSS3 (包含响应式设计)
- JavaScript (ES6+)
- Chart.js (用于数据可视化)
- 使用localStorage进行数据本地存储

## 兼容性
- 桌面端：Chrome、Firefox、Safari、Edge
- 移动端：iOS Safari、Android Chrome
- 响应式设计，适配各种屏幕尺寸

## 数据安全
- 所有数据均保存在本地浏览器中
- 不上传到任何服务器
- 清除浏览器数据将导致记录丢失，请定期备份重要数据
